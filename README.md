# vibe-check
VibeCheck: VibeCoding efficiency monitor for Claude Code. Parses transcript JSONL to compute 6 metrics — TER, TTL, CHR, TTR, IPT, RCS. Two modes: full dashboard report or silent threshold-based watch alerts.        VibeCheck 是 VibeCoding 效率监控技能。解析对话 JSONL 计算 TER、TTL、CHR、TTR、IPT、RCS 六项指标。支持完整报告与静默巡检两种模式，帮助减少 Token 浪费、维持高效编码会话。
